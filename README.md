# harmonica-session-review (archived)

**This skill has been folded back into [harmonica-chat](https://github.com/harmonicabot/harmonica-chat) as a `review` lifecycle subcommand (v2.11.0+).** This repo is preserved as a read-only archive; do not install.

To review a Harmonica session, install harmonica-chat and run:

```
/harmonica-chat review "Q1 retro"
```

See https://github.com/harmonicabot/harmonica-chat for installation.

---

## Why

The skill briefly lived as a standalone repo (2026-05-23) before we recognized the duplication: harmonica-chat already had a lighter `review` subcommand, and maintaining two paths covering the same job — same MCP, same audience, same mental surface — was net cost. The full framework (5 dimensions, structured citation format, pattern aggregation, apply-changes flow) now lives inside harmonica-chat.
